\section{RUNNING THE CODE}

This section briefly explains how to run the code for this assignment. In this assignment, we explore MPI to implement the SUMMA and Cannon's algorithm. Both the SUMMA and Cannon algorithms can be compiled using the \textbf{makefile}.

\begin{enumerate}
    \item The main directory contains \textbf{makefile} and sub-directories \textbf{src}, \textbf{include} and \textbf{docs}.
    \item  To build both SUMMA and Cannon executables, just run -
    \begin{verbatim}
    $ make clean
    $ make
    \end{verbatim}
    \item The above commands produce two executables, \textbf{summa\_rankk\_c} for SUMMA algorithm, and \textbf{cannon\_mm} for Cannon's algorithm.
    \item Running SUMMA code
    \begin{verbatim}
        $ mpirun -np <p> ./summa_rankk_c <N> <k> <seed>
    \end{verbatim}
    where $<p>$ is the total number of MPI processes, $<N>$ is the matrix-size N $\times$ N. $<\text{seed}>$ is the random seed used to generate the test matrices.

    For the correctness test, I run
    \begin{verbatim}
        $ mpirun -np 4  ./summa_rankk_c 12 1 12345
        $ mpirun -np 9  ./summa_rankk_c 12 1 12345
        $ mpirun -np 16 ./summa_rankk_c 12 1 12345
    \end{verbatim}
    where, \textbf{-np 4} corresponds to $2 \times 2$ processor grid, \textbf{-np 9} corresponds to $3 \times 3$ processor grid and \textbf{-np 16} corresponds to $4 \times 4$ processor grid.
    \item Running Cannon code
    \begin{verbatim}
        $ mpirun -np <p> ./cannon_mm <N> <seed>
    \end{verbatim}
    where $<p>$ is the total number of MPI processes, $<N>$ is the matrix-size N x N. $<\text{seed}>$ is the random seed used to generate the test matrices.
    For the correctness test, I run
    \begin{verbatim}
        $ mpirun -np 4  ./cannon_mm 12 12345
        $ mpirun -np 9  ./cannon_mm 12 12345
        $ mpirun -np 16 ./cannon_mm 12 12345
    \end{verbatim}
\end{enumerate}

 
\section{SUMMA algorithm for matrix multiplication}

For the correctness testing, I used $N=12$ and $k=1$,since it satisfies the requirement that the matrix dimensions be divisible by $k\sqrt{p}$ for the $2 \times 2$, $3\times3$ and $4\times4$ processor grids. 

To test correctness, rank 0 generated random matrices $A$ and $B$, computed the serial reference result $C_{ref} = AB$, and packed the matrices into block form. The blocks then were distributed to the MPI processes using MPI\_Scatter. After the parallel SUMMA computation, the local blocks of the parallel results were collected with MPI\_Gather. The serial reference results were also distributed in block form using MPI\_Scatter, so that each process could compute its local maximum absolute error. The global maximum error was then obtained by MPI\_Reduce.
